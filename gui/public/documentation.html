ğŸš€ Getting Started

Prerequisites

Node.js >= 18.0.0
npm >= 9.0.0

Installation

1. Install Dependencies

cd gui
npm install

2. Start Development Server

npm run dev

The application will open at http://localhost:3000

3. Build for Production

npm run build
npm run preview

ğŸ“Š Data Loading

The application loads analysis data from:

  1. /public/data/analysis.json - Real analysis results
  2. /public/data/demo-data.json - Demo data (fallback)

Integrating with Python Analyzer

To connect the GUI with your Python analyzer:

Option 1: Generate JSON

# In your project root
python comprehensive_analyzer.py . -json

# Copy output to GUI
cp output/comprehensive_analysis.json gui/public/data/analysis.json

Option 2: API Integration

Create a backend API endpoint and update the data loading:

const response = await fetch('http://localhost:8000/api/analysis');
const data = await response.json();
setAnalysisData(data);

Edit src/styles/theme.ts to customize colors:

  colors: {
    accent: {
      green: '#00ff9a',  // Change primary accent

Add New Module

  1. Create page: src/pages/NewModule.tsx
  2. Add route in src/App.tsx
  3. Add tab in src/components/layout/TopNav.tsx

ğŸ› Troubleshooting

Demo data not loading

  • Check console for errors
  • Verify public/data/demo-data.json exists
  • Check browser network tab

Charts not rendering

  • Install recharts: npm install recharts
  • Clear cache: rm -rf node_modules/.vite

Build fails

# Clear and reinstall
rm -rf node_modules package-lock.json
npm install
npm run build
